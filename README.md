# 🚀 AI Resume Analyzer + Job Matcher

An AI-powered system that analyzes resumes, matches them with job descriptions, and provides actionable insights using NLP, Machine Learning, and LLMs.

---

## 🔥 Features

- 📄 Resume Parsing (PDF → structured text)
- 🧠 Skill Extraction (NLP + custom skill database)
- 🎯 Semantic Matching (BERT-based similarity)
- 📊 ATS Score Calculation
- ⚠️ Skill Gap Analysis
- 💼 Job Recommendation Engine
- 🤖 AI Email Generator (Gemini API)
- ✨ Resume Improver (LLM-powered rewriting)
<!-- - 🌍 Live Job Fetching (with fallback system) -->
- 📊 Interactive Dashboard (Streamlit)

---

## 🧠 Tech Stack

- Python
- NLP (spaCy)
- Machine Learning (Scikit-learn)
- Sentence-BERT
- Streamlit
- Google Gemini API

---

## 🎯 How It Works

1. Upload resume
2. Input job description
3. System extracts skills & analyzes content
4. Computes semantic similarity
5. Generates:
   - Match Score
   - ATS Score
   - Skill Gap
   - Job Recommendations
   - AI-generated HR email
   - Improved Resume

## 💡 Why This Project?

Most resume analyzers only calculate similarity scores.

This system goes further by:
- Explaining *why* a match is low/high
- Suggesting improvements
- Automating job application workflows
